## 3. 핵심 기능 (구현 완료)

### 3-1. AI 추천 서비스 (핵심)
- "당신에게 가장 맞는 한국 병의원 TOP 3" 추천
- 4단계 퀴즈 (시술/예산/지역/언어지원)
- 추천 3개 + 광고 1개 노출
- 추천 문의 기능

### 3-2. 커뮤니티 (레딧 스타일)
- 카테고리별 게시판 (성형, 피부, 치과, 일반)
- 추천/비추천 (upvote/downvote)
- 댓글, 인기글 정렬
- 시술 후기/리뷰

### 3-3. 병원 검색 (심평원 실데이터 연동 완료)
- 심평원 병원정보서비스 API 실데이터 연동 완료 (79,569개 병원)
- 메인에서 조건 검색 → 페이지 이동 없이 즉시 결과 표시
- 병원명, 지역, 진료과 필터 검색
- 홈페이지 링크 표시

### 3-4. 콘텐츠 코너
- 한국 거리 라이브캠 (5개 채널 — 한강, 반포대교, 롯데타워, 강남, 부산)
- 인기 시술 TOP 5
- FAQ 30개 (8개국 번역)

### 3-5. 사용자 시스템
- Google 소셜 로그인 + 이메일 가입
- 마이페이지 (프로필, 내 글, 내 댓글, AI 추천 이력)
- 계정 탈퇴, 차단/뮤트 기능

### 3-6. 통합 검색
- 병원, 커뮤니티 글, FAQ 한번에 검색
- 언어별 결과 표시

### 3-7. 기타 페이지
- About Us, Contact (문의 폼 → Supabase 저장)
- 이용약관, 개인정보보호, 면책조항 (8개국 번역)
- FAQ, 이용 가이드, 안전 정보, 신고하기
- 404 에러 페이지, 사이트맵, robots.txt

## 4. 데이터 소스

### 4-1. 심평원 병원정보서비스 API (실데이터 연동 완료)

- Endpoint: `hospInfoServicev2/getHospBasisList`
- 연동 상태: **실데이터 연동 완료** (79,569개 병원)
- 메인 페이지 조건 검색 시 페이지 이동 없이 즉시 결과 표시

| 오퍼레이션 | 정보 | 주요 필드 |
|-----------|------|----------|
| getHospBasisList | 병원 기본정보 | 병원명, 종별, 주소, 전화번호, 홈페이지, 의사수, 전문의수, 개설일자 |

- 수집 방식: 하루 1회 Supabase에 저장, 매일 갱신 (예정)
- 제외 필드: 위경도, 진료과목코드, 우편번호, 요양기관번호

### 4-2. Google Places API
- 구글 별점 (1~5점)
- 구글 리뷰 수
- "구글별점 · 리뷰 O건" 으로 표기

### 4-3. 병원 카드 최종 표시

```
🏥 병원명                           [홈페이지 방문]
   종별 · 진료과목
   주소
   📞 전화번호
   👨‍⚕️ 의사 O명 · 🏅 전문의 O명 (과목별)
   🔬 보유장비: CT 1대, 레이저 3대
   📅 개설일: 2010.03.15
   ⭐ 4.5 구글별점 · 리뷰 128건
```
