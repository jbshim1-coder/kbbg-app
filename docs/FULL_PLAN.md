## 3. 핵심 기능 — 기획

### 3-1. AI 추천 서비스 (핵심 서비스)

> "당신에게 가장 맞는 한국 병의원 TOP 3" 추천

- **데이터 소스**:
  - 심평원 병원 데이터 (매월 업데이트)
  - 웹 크롤링 추가 정보 (외국어 홈페이지 여부, 해외 환자 대상 서비스 등)
- **사용자 입력 기반 매칭 (4단계 퀴즈)**:
  - 원하는 시술/진료과목
  - 예산
  - 선호 지역
  - 언어 지원 필요 여부
- **필터 검색**: 전문의 여부, 지역, 원장 수, 진료과목
- **실제 AI 엔진 활용** — 사용자 조건과 병원 데이터를 AI가 분석하여 최적 3개 병원 추천
- 추천 결과에 병원 상세 정보, 강점, 추천 이유 함께 제공
- **추천 3개 + 광고 1개** — AI 추천 병원 3개와 함께 광고 병원 1개를 노출 (광고 표시 명시)
- 사용자가 관리자에게 직접 "이 병원 추천해주세요" 요청 가능 (추천 문의 기능)
- AI 오류/다운 시 → "잠시 후 다시 시도해주세요" 안내 + 인기 병원 목록 대체 표시
- API 호출 제한: 사용자당 1일 추천 요청 횟수 제한 (비용 관리)
- 추천 결과 캐싱으로 AI 서버 부담 감소

## 5. 데이터 소스

### 5-1. 심평원 의료기관 상세정보 API

- **출처**: 건강보험심사평가원 (hira.or.kr) / 공공데이터포털 (data.go.kr)
- **Endpoint**: `https://apis.data.go.kr/B551182/MadmDtlInfoService2.7`
- **수집 방식**: 하루 1회 Supabase에 저장, 매일 갱신
- **API 키 상태**: 발급 완료, 데이터 활성화 대기 중

| 오퍼레이션 | 정보 | 주요 표시 필드 |
|-----------|------|--------------|
| getDtlInfo2.7 | 병원 기본정보 | 병원명, 종별, 주소, 전화번호, 홈페이지, 의사수, 전문의수, 개설일자 |
| getEqpInfo2.7 | 보유 장비정보 | 장비명 (CT, MRI, 레이저 등), 수량 |
| getDgsbjtInfo2.7 | 진료과목 상세 | 진료과목명, 과목별 전문의수 |

- **제외 필드**: 위경도, 진료과목코드, 우편번호, 요양기관번호

### 5-2. Google Places API

- 구글 별점 (1~5점)
- 구글 리뷰 수
- "구글별점 · 리뷰 O건" 으로 표기
- 일일 동기화로 Supabase에 저장

### 5-3. 웹 크롤링 (보충 정보)

- 병원 공식 홈페이지에서 수집:
  - 외국어 지원 여부
  - 해외 환자 전용 서비스
  - 통역 서비스 여부
  - 해외 환자 유치 실적
- robots.txt 및 약관 준수, 공공 API 우선 사용

## 7. 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| 프론트엔드 | Next.js 16 + TypeScript + Tailwind CSS | SSR/SEO, 반응형, 빠른 개발 |
| 백엔드 | Next.js API Routes | 별도 서버 불필요 |
| DB/인증/저장소 | Supabase (PostgreSQL + Auth + Storage) | 관리형, 비개발자 운영 가능 |
| i18n | next-intl (8개국, URL 기반: /en, /ko...) | 8개국 SEO 최적화 |
| AI 추천 | Claude API | 추천 이유 생성 |
| 로그인 | Google OAuth (Supabase Auth) | 글로벌 커버 |
| 호스팅 | Vercel (프론트) + Supabase (백엔드) | 자동 배포, 무료 시작 |
| 코드 관리 | GitHub (jbshim1-coder/kbbg-app) | Vercel 자동 배포 연동 |
| 개발 방식 | Claude Code (AI 코딩) | 비개발자도 운영 가능 |
| 모니터링 | Sentry + Vercel 로그 | 장애 감지 |
| CDN | Vercel Edge Network | 8개국 빠른 로딩 |

## 9. AI 추천 엔진 설계

```
사용자 입력 (4단계 퀴즈)
    ↓
[1단계] 하드 필터 (SQL)
    - 예산, 지역, 시술 종류, 제재 여부
    ↓
[2단계] 점수화
    - 심평원 지표 + 가격 + 최신성 + 사용자 선호
    ↓
[3단계] 의미 매칭
    - 임베딩 유사도로 Top N 정제 (pgvector)
    ↓
[4단계] LLM 재랭킹 (Claude API)
    - Top 3 선정 + 추천 이유 생성
    ↓
[5단계] 광고 삽입
    - 추천 3개 + 광고 1개 (분리된 정책)
    ↓
결과 출력
```

### 심평원 데이터 수집 파이프라인

```
Cloud Scheduler (하루 1회 트리거)
    ↓
API 수집 (증분/체크포인트)
    ↓
Raw 저장 (JSON, 버전 보관)
    ↓
정규화 (병원명/주소/사업자번호 매핑)
    ↓
품질 검증 (누락/이상치/급변 탐지)
    ↓
PostgreSQL 업서트 (이력 보존)
    ↓
결과 리포트 자동 생성
    ↓
실패 시 재시도 + 알림 (이메일)
```

- 게시글/댓글 관리 (승인, 삭제, 숨김)

#### 광고 관리
- AI 추천 광고 병원 등록/관리

- 카테고리 스폰서 병원 설정

- 광고비 청구/결제 관리

#### 분석/통계
- 방문자 통계 (국가별, 시술별, 인기 페이지)
- Google Analytics 연동 대시보드

#### 시스템 관리
- 알림 설정 관리
- 스팸 필터 규칙 설정
- 데이터 백업 상태 확인
- AI 번역 품질 모니터링
- 법적 고지/면책 조항 관리

### 11-3. 스팸/광고 방지 체계

- **AI 자동 필터링** — 스팸/광고 게시글 자동 감지 및 즉시 삭제
- **신규 계정 제한** — 가입 후 24시간 내 게시 불가, 첫 3개 글은 관리자 승인 필요
- **도배 차단** — 동일 IP/계정에서 단시간 다량 게시 시 자동 차단
- **금칙어 시스템** — 광고성 키워드/URL 자동 차단 (8개국 언어별 금칙어 목록)
- **신고 기능** — 사용자 신고 3회 누적 시 게시글 자동 숨김
- **자동 차단** — 신고 누적 5회 이상 계정 자동 정지
- **이미지 필터** — 부적절한 이미지 AI 자동 감지
- **reCAPTCHA** — 회원가입/게시 시 봇 차단
- **블랙리스트** — 차단된 IP/이메일/전화번호 DB 관리

### 11-4. 콘텐츠 업데이트 주기

- 트렌드 리포트, 인기 랭킹: 매일 업데이트

- 의사 영상: 유튜브 조회수 기준 자동 큐레이션 (매일)

### 11-5. 데이터 보관 정책

- 게시글/댓글: 삭제 요청 전까지 보관

### 11-6. 일상 운영 가이드 (비개발자용)

#### 매일
1. 관리자 대시보드 접속

## 12. 법적 준수사항

### 12-1. 사업자 등록

|---------|---------|

| 미국 | CCPA (캘리포니아) |
| 유럽 경유 사용자 | GDPR |
| 기타 국가 | 현지 법령 검토 |

### 12-4. 의료광고법 준수

- 한국 의료법상 전후사진 규제
- 과대광고 금지

- 동의 철회 시 즉시 삭제

### 12-6. 개인정보 삭제 요청 (GDPR 삭제권)

- 사용자 요청 시 개인정보 완전 삭제 프로세스
- 삭제 요청 처리 기한 준수 (30일 이내)

### 12-7. 쿠키 동의

- GDPR 대응 쿠키 동의 배너
- 필수/선택 쿠키 구분
- 국가별 쿠키 정책 대응

### 12-8. 크롤링 법적 정책

- robots.txt 및 약관 준수
- 공공 API 우선 사용, 크롤링은 보충 용도
- 차단 대응 정책 수립

### 12-9. 사용자 콘텐츠 저작권

- 사용자가 올린 글/사진의 저작권은 사용자에게 있음
- 플랫폼은 서비스 내 노출/홍보 목적으로 이용할 수 있는 권리 부여 (이용약관에 명시)
- 사용자 요청 시 콘텐츠 삭제 보장

- Google SGE, Bing Copilot 등 AI 검색엔진에서 답변으로 인용되도록 최적화

### 13-3. AIEO (AI Engine Optimization — AI 엔진 최적화)

- ChatGPT, Claude, Gemini 등 AI 챗봇이 답변 시 우리 사이트를 인용/추천하도록 최적화

- 고유 데이터 생산 — AI 추천 결과, 가격 비교표, 트렌드 리포트
- 명확한 브랜드명 반복 — "K-Beauty Buyers Guide에 따르면..." 형태로 인용 유도
- 시술별 전문 가이드 페이지 — 각 시술에 대한 심층 정보 (AI 학습 소스로 활용)
- E-E-A-T 강화 — 경험(Experience), 전문성(Expertise), 권위(Authority), 신뢰(Trust)
- 외부 링크/인용 확보 — 의료관광 관련 사이트에서 백링크 확보
