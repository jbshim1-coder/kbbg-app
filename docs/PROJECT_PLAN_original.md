## 3. 핵심 기능

### 3-1. AI 추천 서비스 (핵심 서비스)

> "당신에게 가장 맞는 한국 병의원 TOP 3" 추천

- **데이터 소스**:
  - 심평원 병원 데이터 (매월 업데이트)
  - 웹 크롤링 추가 정보 (외국어 홈페이지 여부, 해외 환자 대상 서비스 등)
- **사용자 입력 기반 매칭**:
  - 원하는 시술/진료과목
  - 예산
  - 선호 지역
  - 언어 지원 필요 여부
- **필터 검색**:
  - 전문의 여부
  - 지역
  - 원장 수
  - 진료과목
- **실제 AI 엔진 활용** — 사용자 조건과 병원 데이터를 AI가 분석하여 최적 3개 병원 추천
- 추천 결과에 병원 상세 정보, 강점, 추천 이유 함께 제공
- **추천 3개 + 광고 1개** — AI 추천 병원 3개와 함께 광고 병원 1개를 노출 (광고 표시 명시)
- 사용자가 관리자에게 직접 "이 병원 추천해주세요" 요청 가능 (추천 문의 기능)

### 3-2. 커뮤니티 (레딧 스타일)

### 3-3. Q&A 게시판

- 자유 질문/답변

### 3-4. 콘텐츠 코너 (체류시간 증가 목적)

- 한국 의사 추천 영상 — 유튜브 조회수 기준 자동 큐레이션 (시술 설명, 팁 등)
- 한국 뷰티 인플루언서 큐레이션 코너 (기존 콘텐츠 선별 소개)
- 영상/콘텐츠에 자막 또는 번역 제공 (7개국 언어)
- **전후 변화 갤러리** — 시술별 Before/After 사진 모음 (로그인 사용자만 열람 가능)

## 5. 트래픽 & 성장 전략

### 5-1. SEO + AEO + AIEO 전략

#### 기존 SEO (검색엔진 최적화)
- 7개국 주요 검색엔진 최적화 (Google, Baidu, Yandex, Yahoo Japan 등)
- 국가별 언어로 시술 키워드 페이지 생성
- 검색엔진별 메타태그, 구조화 데이터 적용

#### AEO (Answer Engine Optimization — AI 답변 최적화)
- Google SGE, Bing Copilot 등 AI 검색엔진에서 답변으로 인용되도록 최적화
- FAQ 형식 콘텐츠 — "한국 코성형 비용은?", "한국 피부과 추천은?" 등 질문-답변 구조
- Schema.org FAQPage, HowTo, MedicalClinic 구조화 데이터 적용
- 명확하고 간결한 답변 형태의 콘텐츠 작성 (AI가 인용하기 쉬운 구조)
- 통계/수치 포함 — "한국 코성형 평균 비용: $3,000~$5,000" 등

#### AIEO (AI Engine Optimization — AI 엔진 최적화)
- ChatGPT, Claude, Gemini 등 AI 챗봇이 답변 시 우리 사이트를 인용/추천하도록 최적화
- 권위 있는 콘텐츠 — 심평원 공인 데이터 기반 (신뢰도 높은 출처)
- 고유 데이터 생산 — AI 추천 결과, 가격 비교표, 트렌드 리포트 (다른 곳에 없는 데이터)
- 명확한 브랜드명 반복 — "K-Beauty Buyers Guide에 따르면..." 형태로 인용 유도
- 시술별 전문 가이드 페이지 — 각 시술에 대한 심층 정보 (AI 학습 소스로 활용)
- E-E-A-T 강화 — 경험(Experience), 전문성(Expertise), 권위(Authority), 신뢰(Trust)
- 외부 링크/인용 확보 — 의료관광 관련 사이트에서 백링크 확보

### 5-2. SNS 연동 & 바이럴

### 7-4. 콘텐츠 업데이트 주기

- 트렌드 리포트, 인기 랭킹, 콘텐츠 큐레이션: **매일 업데이트**

### 7-5. 스팸/광고 방지 (강한 대응 체계)

- **AI 자동 필터링** — 스팸/광고 게시글 자동 감지 및 즉시 삭제
- **신규 계정 제한** — 가입 후 24시간 내 게시 불가, 첫 3개 글은 관리자 승인 필요
- **도배 차단** — 동일 IP/계정에서 단시간 다량 게시 시 자동 차단
- **금칙어 시스템** — 광고성 키워드/URL 자동 차단 (7개국 언어별 금칙어 목록)
- **신고 기능** — 사용자 신고 3회 누적 시 게시글 자동 숨김
- **자동 차단** — 신고 누적 5회 이상 계정 자동 정지
- **이미지 필터** — 부적절한 이미지 AI 자동 감지
- **reCAPTCHA** — 회원가입/게시 시 봇 차단
- **블랙리스트** — 차단된 IP/이메일/전화번호 DB 관리

## 8. 법적 준수사항

### 8-0. 사업자 등록

- 법인 사업자로 운영
- 의료 중개 관련 사업자 등록 확인 필요

### 8-1. 면책 조항

- "AI 추천은 의료 행위가 아닌 참고 정보" 명시
- 플랫폼은 병원과 환자 간의 중개 역할이며, 치료 결과에 대한 책임을 지지 않음

### 8-2. 개인정보보호 정책

- 7개국 각각의 개인정보보호법 대응
  - 중국: PIPL (개인정보보호법)
  - 미국: CCPA (캘리포니아)
  - 유럽 경유 사용자: GDPR
  - 기타 국가: 현지 법령 검토

### 8-3. 의료광고법 준수

- 한국 의료법상 전후사진 규제
- 과대광고 금지
- 환자 후기 게시 시 가이드라인 적용

### 8-4. 크롤링 법적 정책

- robots.txt 및 약관 준수
- 공공 API 우선 사용, 크롤링은 보충 용도
- 차단 대응 정책 수립

### 8-5. 전후사진 초상권/동의 관리

- 전후사진 업로드 시 초상권 동의서 필수
- 동의 기록 추적 시스템 (consent_records)
- 동의 철회 시 즉시 삭제

### 8-6. 개인정보 삭제 요청 처리

- GDPR 삭제권(Right to Erasure) 대응
- 사용자 요청 시 개인정보 완전 삭제 프로세스
- 삭제 요청 처리 기한 준수 (30일 이내)

### 8-7. 이용약관/서비스 약관

- 서비스 이용약관 (7개국 언어)
- 커뮤니티 이용 규칙
- 광고 표시 관련 고지

### 8-8. 쿠키 동의

- GDPR 대응 쿠키 동의 배너
- 필수/선택 쿠키 구분
- 국가별 쿠키 정책 대응

## 11. 기술 스택

> 비개발자(마케터)가 AI(Claude Code)를 이용해 개발하는 프로젝트.
> 모든 기술 선택은 "관리가 쉬운가?"를 최우선으로 판단.

### 11-1. 개발 방식

- **Claude Code (AI)** 가 코드를 작성하고, 사용자는 요구사항만 전달
- 개발자 없이 AI로 개발 → 코드를 직접 수정할 필요 없음
- 문제 발생 시 Claude Code에 "이거 고쳐줘"라고 요청

### 11-2. 호스팅 & 배포

- **Vercel** (프론트엔드) — 코드 올리면 자동 배포, 무료 시작
- **Supabase** (백엔드/DB) — 데이터베이스, 회원가입, 파일 저장 올인원
- 비용: 초기 무료 → 트래픽 증가 시 유료 전환 (월 $20~50 예상)

### 11-3. 도메인 & 이메일

- 도메인 구매: kbeautybuyersguide.com
- 도메인 연결: Vercel에 도메인 연결 (Claude Code가 안내)
- 비즈니스 이메일: info@kbeautybuyersguide.com (Google Workspace 또는 무료 포워딩)

### 11-4. 성능 최적화

- 이미지 최적화: 전후사진 자동 압축/리사이즈
- CDN: Vercel Edge Network로 7개국 빠른 로딩
- Core Web Vitals 최적화 (Google 검색 순위에 영향)
- AI 추천 결과 캐싱 (동일 조건 재요청 시 빠른 응답)
- 번역 결과 캐싱 (반복 번역 비용 절감)

### 11-5. AI 추천 장애 대응

- AI 오류/다운 시 → "잠시 후 다시 시도해주세요" 안내 + 인기 병원 목록 대체 표시
- API 호출 제한: 사용자당 1일 추천 요청 횟수 제한 (비용 관리)
- 추천 결과 캐싱으로 AI 서버 부담 감소

### 11-6. 다국어 시간대/통화

- 7개국 날짜 형식 자동 대응 (MM/DD vs DD/MM 등)
- 가격 비교 시 각국 통화 자동 환산 (USD, CNY, JPY, RUB, VND, THB, MNT)
- 환율 자동 업데이트

### 11-7. SSL/보안

- HTTPS (SSL 인증서) — Vercel에서 자동 제공 (무료)
- 사용자 데이터 암호화 — Supabase 기본 제공
- DDoS 기본 방어 — Vercel/Supabase 기본 제공
- 정기 보안 업데이트

### 11-5. 중국 접속

- 중국 본토는 방화벽(GFW)으로 접속 어려울 수 있음
- 현재는 별도 대응 없이 진행
- 중국 사용자는 VPN 사용 또는 향후 중국 서버 검토

- 소비자 결제 기능 없음 (1단계는 무료 서비스)
- Phase 1 MVP 완성 즉시 오픈

### 15-1. KPI / 목표 지표

| 지표 | 런칭 후 3개월 | 런칭 후 6개월 | 런칭 후 12개월 |
|------|-------------|-------------|-------------|
| 월 방문자 | 1,000명 | 5,000명 | 20,000명 |
| 가입 회원 | 200명 | 1,000명 | 5,000명 |
| AI 추천 이용 | 100건/월 | 500건/월 | 2,000건/월 |
| 게시글 | 50건/월 | 200건/월 | 500건/월 |
| 광고 병원 | - | 5개 | 20개 |

> KPI는 운영 상황에 따라 조정

### 15-2. 사용자 콘텐츠 저작권

- 사용자가 올린 글/사진의 저작권은 사용자에게 있음
- 플랫폼은 서비스 내 노출/홍보 목적으로 이용할 수 있는 권리 부여 (이용약관에 명시)
- 사용자 요청 시 콘텐츠 삭제 보장

### 15-3. 시즌 전략

| 시기 | 전략 |
|------|------|
| 봄 (3~5월) | 성수기 — 피부 시술 콘텐츠 강화, 광고 집중 |
| 여름 (6~8월) | 비수기 — 가격 비교, 할인 정보 콘텐츠 |
| 가을 (9~11월) | 성수기 — 성형 시술 콘텐츠 강화, 캠페인 |
| 겨울 (12~2월) | 회복기 콘텐츠 — 시술 후 관리, 후기 유도 |

### 15-4. 온보딩 가이드

- 첫 방문 시 "3단계로 나에게 맞는 병원 찾기" 안내 팝업
- 1단계: 퀴즈/자가진단
- 2단계: AI 추천 결과 확인
- 3단계: 커뮤니티 후기 탐색
- "다시 보지 않기" 옵션

### 15-5. 소셜 프루프

- 사이트에 실시간 표시: "현재 회원 O명", "리뷰 O건", "추천 O건"
- 국가별 사용자 분포 표시
- "오늘 O명이 추천 받았습니다" 실시간 카운터

### 15-6. 점검/유지보수 페이지

- 사이트 업데이트/점검 시 다국어 "점검 중" 안내 페이지
- 예상 복구 시간 표시
- 문의 이메일 (help@2bstory.com) 안내

## 16. 향후 계획

- 모바일 앱: 현재 계획 없음, 향후 트래픽/수요에 따라 검토
- 병원 상세 페이지 확장: 시술 목록, 리뷰 모음, 예약 기능
- 병원 직접 등록 기능: 병원/에이전시가 직접 정보 입력
- 다국어 고객 지원 챗봇: AI 기반 실시간 상담
- 소비자 결제 기능: 향후 프리미엄 서비스 도입 시 검토
